🔧 Git & GitHub for DevOps
This repository contains my Git & GitHub notes and practicals used in DevOps.
It covers:
•	Source Code Management (CVCS vs DVCS)
•	Git basics, setup, and workflow (add, commit, push, pull)
•	Git stages (Workspace, Staging, Repository)
•	Branching, merging, stash, reset, revert
•	GitHub usage (clone, remote repo, collaboration)
•	Important concepts like .gitignore, tags, and logs
💡 Purpose: To understand version control and manage code efficiently in real-world DevOps projects.
